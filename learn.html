ECB Mode
(Electronic Codebook)

How it works:
Encrypts each block independently:
Ci = Ek(Pi)
Danger Zone:
  • Shows image patterns
  • No diffusion
  • Block reordering
Attacks:
  • Frequency analysis
  • Block substitution
  • Pattern recognition


CBC Mode
(Cipher Block Chaining)

How it works:
Chains blocks together:
Ci = Ek(Pi ‚äï Ci-1)
Watch Out:
  • Needs random IV
  • Padding oracles
  • No integrity
Technical Notes:
  • Error affects 2 blocks
  • Sequential encryption
  • Parallel decryption


GCM Mode
(Galois/Counter Mode)

How it works:
CTR + Authentication:
Ci = Pi ‚äï Ek(Nonce||Counteri)
Why It's Awesome:
  • Authenticated
  • No padding
  • HW accelerated
Advantages:
  • Unique nonce required
  • Parallel processing
  • TLS 1.3 standard
